Docker Assessment

Before you begin...

  • Ensure with your instructor that your instance has been reset.
  • Make sure you can login to your instance using your account and password (see here for reference).

Tasks

Task 1: Introduction to Docker

  1. Install Docker

    Task: Install Docker on a the instance provided.

    Validation: Docker should be installed and the Docker daemon should be running.

  2. Run a Hello World Container

    Task: Pull the hello-world image from Docker Hub and run it.

    Validation: The output should display a message from the Hello World container.

Task 2: Docker Images and Containers

  1. Pull and Run an Nginx Container

    Task: Pull the nginx image and run it, exposing port 80 and naming it docker_nginx.

    Validation: The Nginx welcome page should be accessible from the instance IP and named docker_nginx.

  2. Inspect Nginx Container

    Task: Inspect the running Nginx container to view its configuration details. Save a copy of it in ~/docker_inspect.txt

    Validation: The container's configuration details should be present in ~/docker_inspect.txt.

Task 3: Docker Volumes

  1. Create and Attach a Volume

    Task: Create a Docker volume named nginx_data and attach it to an Nginx container.

    Validation: The volume should be listed under Docker volumes, and Nginx should use it.

  2. Persist Data Using Volumes

    Task: Create an HTML file in the nginx_data volume and verify it's served by Nginx.

    Validation: Accessing the Nginx server should display the custom HTML page.

Task 4: Docker Networks

  1. Create a Custom Docker Network

    Task: Create a custom bridge network named my_network.

    Validation: The network should be listed under Docker networks.

  2. Run Containers on Custom Network

    Task: Run an Nginx and a MySQL container on my_network.

    Validation: Both containers should be able to communicate within the custom network.

  3. Connect Containers Across Networks

    Task: Connect an existing Nginx container to the my_network while it's running.

    Validation: Nginx container should be connected to my_network.
